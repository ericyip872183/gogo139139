# 项目概览

> 盼蕾平台 — 中医/医学 SaaS 一体化教学考试平台

## 定位

考试母本系统（所有模块共用核心）+ 可插拔专业模块（独立界面）+ 硬件实操对接 + 多维度统一评分。

## 技术栈

| 层 | 技术 |
|----|------|
| 前端 | Vue 3 + TypeScript + Vite 5 + Element Plus 2.5 + Pinia + Vue Router 4 |
| 后端 | NestJS 10 + Prisma 5.22 + MySQL 8.0 + Redis 7 + JWT + bcrypt |
| 硬件 | Web Serial API + Web Bluetooth API（Chrome 89+） |
| 构建 | pnpm workspace monorepo |

## 端口

| 服务 | 开发 | 生产 |
|------|------|------|
| 前端 | 5173 | 80（nginx） |
| 后端 | 3002 | 3000（容器内） |
| MySQL | 3306 | 3306（内部） |
| Redis | 6379 | 6379（内部） |

## 角色体系

5种角色，层级从高到低：

```
SUPER_ADMIN（超级管理员，平台运营）
  └─ TENANT_ADMIN（机构管理员）
       └─ CLASS_ADMIN（班级管理员）
            └─ TEACHER（教师）
                 └─ STUDENT（学生）
```

只能管理层级低于自己的用户。

## 后端模块（11个）

| 模块 | 路径 | 核心功能 |
|------|------|----------|
| auth | modules/auth | 登录、JWT、验证码、密码重置 |
| users | modules/users | 用户CRUD、批量导入、软删除 |
| organizations | modules/organizations | 树形组织架构（4级） |
| questions | modules/questions | 题库、分类、媒体资源、AI导入 |
| papers | modules/papers | 试卷模板管理 |
| exams | modules/exams | 考试管理、参与者、发布/取消 |
| exam-room | modules/exam-room | 在线答题、切屏检测、自动交卷 |
| scores | modules/scores | 成绩查询、统计、Excel导出 |
| score-tables | modules/score-tables | 评分表、打分记录 |
| admin | modules/admin | 超管后台（机构/模块/用户管理） |
| ai | modules/ai | AI对话、OCR、图片生成、配额管理 |

## 前端页面（16个）

LoginView / DashboardView / OrganizationView / UsersView / QuestionsView / PapersView / ExamsView / ExamRoomView / ScoresView / ScoreTablesView / ScoreJudgeView / AdminView / AiPlatformView / TenantServiceCenterView / MyExamsView / ProfileView / HardwareDebugPanel

均以 `TCM_` 前缀：CONSTITUTION / EAR / MERIDIAN / FOURDIAG / ACUPUNCTURE / GUASHA / MASSAGE / ANATOMY
