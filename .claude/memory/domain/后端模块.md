# 后端模块

## 认证模块（auth）

**路径**：`modules/auth/`

JWT Payload：`{ sub: userId, tenantId, role }`

登录接口：
- `POST /api/auth/login` — 账号密码登录（需传 tenantCode）
- `POST /api/auth/login-by-code` — 验证码登录
- `POST /api/auth/send-code` — 发送验证码
- `POST /api/auth/reset-password` — 重置密码
- `GET /api/auth/tenants/:code` — 查询机构信息
- `GET /api/auth/profile` — 获取当前用户信息
- `GET /api/auth/my-modules` — 获取已购模块

## 用户模块（users）

**路径**：`modules/users/`

软删除实现：`isDeleted / deletedAt / deletedBy` 字段，删除后数据保留。

主要接口：
- `GET /api/users` — 列表（支持 keyword/organizationId/role/page/pageSize）
- `POST /api/users` — 创建（权限：只能创建层级低于自己的角色）
- `PATCH /api/users/:id` — 更新
- `DELETE /api/users/:id` — 软删除
- `DELETE /api/users/:id/force` — 物理删除
- `POST /api/users/:id/restore` — 恢复软删除用户
- `POST /api/users/import-excel` — Excel批量导入
- `GET /api/users/export` — Excel导出
- `PATCH /api/users/batch-status` — 批量激活/停用
- `PATCH /api/users/batch-password` — 批量设置密码
- `GET /api/users/deleted/list` — 已删除用户列表
- `PATCH /api/users/me/password` — 修改自己密码

## 题库模块（questions）

**路径**：`modules/questions/`

题目类型：`SINGLE`（单选）/ `MULTIPLE`（多选）/ `JUDGE`（判断）/ `FILL`（填空）

难度：`EASY` / `MEDIUM` / `HARD`

主要接口：
- `GET/POST /api/questions/categories/tree` — 分类树
- `GET/POST/PATCH/DELETE /api/questions` — 题目CRUD
- `DELETE /api/questions/batch` — 批量删除
- `POST /api/questions/import-excel` — Excel导入
- `GET /api/questions/export` — Excel导出
- `POST /api/questions/:id/media` — 上传媒体资源
- `POST /api/questions/ai-import/upload` — AI导入上传文件
- `GET /api/questions/ai-import/tasks/:id` — 查询导入任务进度
- `POST /api/questions/ai-import/tasks/:id/confirm` — 确认导入
- `POST /api/questions/ai-import/tasks/:id/skip` — 跳过题目

AI导入流程：上传文件 → 异步解析（火山引擎Responses API）→ 人工校对 → 批量入库

## 考试模块（exams + papers + exam-room）

考试状态：`DRAFT → PUBLISHED → ONGOING → ENDED / CANCELLED`

**试卷**（papers）：
- `GET/POST/PATCH/DELETE /api/papers` — 试卷CRUD

**考试**（exams）：
- `POST /api/exams/:id/publish` — 发布
- `POST /api/exams/:id/cancel` — 取消
- `POST /api/exams/:id/clone` — 克隆
- `POST /api/exams/:id/participants` — 添加参与者（支持按用户ID或组织ID）

**答题**（exam-room）：
- `GET /api/exam-room/:id` — 进入考试（返回题目，不含答案）
- `POST /api/exam-room/:id/answer` — 保存答案（实时）
- `POST /api/exam-room/:id/switch` — 记录切屏（超限自动交卷）
- `POST /api/exam-room/:id/submit` — 提交答卷

## 成绩模块（scores）

- `GET /api/scores/exam/:id` — 考试成绩列表
- `GET /api/scores/exam/:id/stats` — 成绩统计
- `GET /api/scores/exam/:id/detail/:userId` — 答题详情
- `GET /api/scores/exam/:id/export` — Excel导出
- `GET /api/scores/my` — 学生个人成绩

## 评分表模块（score-tables）

支持加分制（ADD）和减分制（MINUS）。

- `GET/POST/PATCH/DELETE /api/score-tables` — 评分表CRUD
- `POST /api/score-tables/records` — 提交打分
- `GET /api/score-tables/:id/records` — 查看打分记录
- `POST /api/score-tables/:id/convert-to-questions` — 转换为题目

## 超管模块（admin）

仅 SUPER_ADMIN 可访问。

- `GET/POST/PATCH /api/admin/tenants` — 机构管理
- `GET/POST /api/admin/modules` — 模块管理
- `POST/DELETE /api/admin/tenants/:id/modules` — 机构模块授权
- `GET /api/admin/stats` — 平台统计
- `GET /api/admin/operation-logs` — 操作日志

## AI模块（ai）

**架构**：AiProvider（服务商）→ AiModel（模型）→ TenantAiModel（机构配置）→ AiUsage（使用日志）

**用户接口**（TEACHER+）：
- `POST /api/ai/chat` — 模拟病人对话
- `POST /api/ai/ocr` — OCR识别
- `GET /api/ai/usage` — 使用统计

**超管接口**（SUPER_ADMIN）：
- `GET/POST/PUT/DELETE /api/ai/admin/providers` — 服务商管理
- `GET/POST/PUT/DELETE /api/ai/admin/models` — 模型管理
- `GET/POST/DELETE /api/ai/admin/tenant-models` — 机构模型配置
- `GET/POST /api/ai/admin/quotas` — 配额管理
- `GET /api/ai/admin/stats/platform` — 平台统计
- `POST /api/ai/admin/models/:id/test` — 测试模型连接
- `POST /api/ai/admin/models/generate-image` — 图片生成
- `POST /api/ai/admin/models/:id/chat-stream` — 流式对话（SSE）
