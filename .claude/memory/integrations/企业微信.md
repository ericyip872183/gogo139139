| 项目 | 值 |
|------|-----|
| 服务商 | 腾讯企业微信 |
| API端点 | `https://qyapi.weixin.qq.com/cgi-bin` |
| 鉴权 | access_token（corpid + secret 换取，有效期2小时） |

## 环境变量

```bash
WECHAT_WORK_CORPID=your_corpid
WECHAT_WORK_SECRET=your_secret
WECHAT_WORK_AGENT_ID=your_agent_id
WECHAT_WORK_CALLBACK_URL=https://your-domain.com/api/wechat-work/callback
```

## 核心接口

```
GET  /gettoken                  — 获取access_token（corpid+secret）
GET  /auth/getuserinfo          — 通过code获取用户身份
GET  /department/list           — 获取部门列表
GET  /user/list                 — 获取部门成员
POST /message/send              — 发送应用消息
```

## 注意事项

- access_token 有效期2小时，必须用 Redis 缓存（建议缓存1.5小时）
- 不得在前端暴露 corpid 和 secret
- 回调URL必须验证签名
- 消息推送频率限制：每分钟不超过1000次
- 本地开发需要内网穿透（如 ngrok）

## 登录流程

```
前端重定向到企业微信授权页
  → 用户扫码确认
  → 企业微信回调携带 code
  → 后端用 code 换取用户信息
  → 创建JWT Token返回前端
```

## 错误码

| 错误码 | 含义 |
|--------|------|
| 40001 | 不合法的secret |
| 40014 | 不合法的access_token |
| 42001 | access_token已过期 |
| 60011 | 没有权限 |
| 60012 | 用户不在应用可见范围 |
