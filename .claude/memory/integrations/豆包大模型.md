## SDK使用（推荐）

```typescript
import OpenAI from 'openai'  // npm install openai

const client = new OpenAI({
  baseURL: 'https://ark.cn-beijing.volces.com/api/v3',
  apiKey: process.env.ARK_API_KEY,
})
```

## Chat API（对话）

```typescript
// 标准对话
const response = await client.chat.completions.create({
  model: 'ep-20260315163053-ns7kz',
  messages: [
    { role: 'system', content: '你是助手' },
    { role: 'user', content: '你好' }
  ]
})

// 流式对话
const stream = await client.chat.completions.create({
  model: 'ep-20260315163053-ns7kz',
  messages: [{ role: 'user', content: '你好' }],
  stream: true
})
for await (const chunk of stream) {
  process.stdout.write(chunk.choices[0].delta.content ?? '')
}
```

## Responses API（文档/图片理解）

用于 AI 导入题库时解析 PDF/Word/图片文件。

### 三种文件输入方式

**方式一：File ID（推荐，≤512MB）**
```typescript
// 先上传文件
const file = await client.files.create({
  file: fs.createReadStream('/path/to/doc.pdf'),
  purpose: 'user_data',
})

// 再调用Responses API
const response = await client.responses.create({
  model: 'doubao-seed-2-0-lite-260215',
  input: [{
    role: 'user',
    content: [
      { type: 'input_file', file_id: file.id },
      { type: 'input_text', text: '请提取文档中的题目' }
    ]
  }]
})
```

**方式二：File URL（需公网可访问，≤50MB）**
```typescript
{ type: 'input_file', file_url: 'https://example.com/doc.pdf' }
```

**方式三：Base64（≤50MB）**
```typescript
const base64 = fs.readFileSync('/path/to/doc.pdf', 'base64')
{ type: 'input_file', file_data: `data:application/pdf;base64,${base64}`, filename: 'doc.pdf' }
```

### 图片输入

```typescript
// File ID方式
{ type: 'input_image', file_id: 'xxx' }

// URL方式
{ type: 'image_url', image_url: { url: 'https://...' } }
```

## 常见错误（必读）

### 错误1：文本类型字段写错

```json
// ❌ 错误
{ "type": "text", "text": "内容" }

// ✅ 正确
{ "type": "input_text", "text": "内容" }
```

### 错误2：文档用了Chat API而非Responses API

```typescript
// ❌ 错误：文档理解不能用 chat.completions
await client.chat.completions.create({ messages: [...] })

// ✅ 正确：文档理解用 responses
await client.responses.create({ input: [...] })
```

### 错误3：响应数据结构解析错误

Responses API 返回结构：
```typescript
// ✅ 正确解析方式
if (Array.isArray(data.output)) {
  const messageOutput = data.output.find((item: any) =>
    item.type === 'message' && item.content?.[0]?.text
  )
  const text = messageOutput?.content?.[0]?.text
}
```

### 错误4：图片用了input_file而非input_image

```json
// ❌ 错误：图片不能用 input_file
{ "type": "input_file", "file_id": "xxx" }

// ✅ 正确：图片用 input_image
{ "type": "input_image", "file_id": "xxx" }
```
