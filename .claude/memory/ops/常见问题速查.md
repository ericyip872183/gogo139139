# 常见问题速查

## 容器重启规则

| 场景 | 命令 |
|------|------|
| 修改后端代码 | `docker restart panlei-backend` |
| 修改前端代码 | `docker restart panlei-frontend` |
| 修改依赖/环境变量 | `docker-compose up -d` |
| 端口被占用 | `docker-compose down && docker-compose up -d` |

## 后端启动失败

**数据库连接失败**
```
PrismaClientInitializationError: Can't reach database server at `mysql:3306`
```
原因：MySQL容器未就绪。解决：等10秒后 `docker restart panlei-backend`

**端口被占用**
```
Error: Bind for 0.0.0.0:3002 failed: port is already allocated
```
```bash
netstat -ano | findstr :3002   # 找到PID
taskkill /F /PID <PID>
```

**环境变量缺失**
```
Error: Missing environment variable: DATABASE_URL
```
检查 `.env` 文件是否存在，`docker-compose down && docker-compose up -d` 重新加载。

## 前端启动失败

**后端代理失败**
```
[vite] http proxy error: /api
Error: connect ECONNREFUSED 127.0.0.1:3002
```
原因：后端未启动。`docker restart panlei-backend`

## API错误

| 错误码 | 原因 | 解决 |
|--------|------|------|
| 401 | Token未携带/过期 | 重新登录 |
| 403 | 角色权限不足 | 检查用户角色 |
| 404 | 接口路径错误 | 检查是否有 `/api` 前缀 |
| 500 | 业务逻辑错误 | `docker logs panlei-backend -f --tail 100` |

## 数据库问题

**查询返回空数据**：检查 tenantId 是否正确（从JWT提取，不是请求参数）

**Prisma同步失败**：
```bash
docker exec panlei-backend sh -c "cd /app/packages/backend && prisma db push"
```

**登录密码错误**：见 `domain/部署运维.md` 的重置密码命令

## 日志查看

```bash
docker logs panlei-backend -f --tail 100   # 后端
docker logs panlei-frontend -f --tail 50   # 前端
docker logs panlei-mysql -f --tail 50      # MySQL
docker ps -a                               # 所有容器状态
```

## 前端页面问题

- **页面空白**：F12 → Console 查看红色错误
- **数据不显示**：F12 → Network 检查API请求状态和响应
- **样式错乱**：检查 Element Plus 是否正确引入
