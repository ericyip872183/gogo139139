## 一、角色权限划分

| 角色 | AI 相关权限 | 界面入口 |
|------|------------|----------|
| **超级管理员** | ✅ 添加/编辑 AI 服务商<br>✅ 配置模型列表及 EP<br>✅ 设置各机构可用模型<br>✅ 分配机构配额<br>✅ 查看全平台统计<br>✅ 设置预警阈值 | 超管后台 → AI 平台管理 |
| **机构管理员** | ✅ 查看本机构可用服务<br>✅ 查看配额使用情况<br>✅ 申请增加配额<br>✅ 为本机构充值<br>✅ 查看本机构统计<br>❌ **不可修改模型配置** | AI 服务中心 |
| **教师/学生** | ✅ 使用 AI 功能（虚拟病人/题目导入等）<br>✅ 查看个人使用记录<br>❌ 无配置权限 | 无独立入口，嵌入功能模块 |

### 6.1 EP 处理逻辑

```typescript
// packages/backend/src/modules/ai/ai.service.ts

/**
 * 构建完整端点 URL
 */
function buildEndpoint(model: AiModel, provider: AiProvider): string {
  if (model.isEp) {
    // EP 模式：https://{ep}.cn-beijing.volces.com/api/v3/chat/completions
    if (model.modelId.startsWith('ep-')) {
      return `https://${model.modelId}.cn-beijing.volces.com/api/v3/chat/completions`
    }
    if (model.modelId.startsWith('http')) {
      return model.modelId
    }
  }
  // 标准模式：{baseUrl}/chat/completions
  return `${provider.baseUrl}/chat/completions`
}

/**
 * 获取机构某场景的默认模型
 */
async function getTenantDefaultModel(
  tenantId: string,
  scene: string
): Promise<{ model: AiModel; provider: AiProvider; endpoint: string }> {
  // 查询机构该场景的默认模型
  const tenantModel = await this.prisma.tenantAiModel.findFirst({
    where: { tenantId, scene, isDefault: true, isEnabled: true },
    include: {
      model: { include: { provider: true } }
    },
  })

  if (!tenantModel) {
    throw new BadRequestException(`该场景未配置默认模型：${scene}`)
  }

  return {
    model: tenantModel.model,
    provider: tenantModel.model.provider,
    endpoint: buildEndpoint(tenantModel.model, tenantModel.model.provider),
  }
}
```

### 6.2 配额扣减逻辑

```typescript
/**
 * 扣减配额并检查预警
 */
async function deductQuota(
  tenantId: string,
  providerId: string,
  tokensUsed: number
) {
  // 扣减机构配额
  await this.prisma.tenantAiQuota.update({
    where: { tenantId_providerId: { tenantId, providerId } },
    data: { usedQuota: { increment: tokensUsed } },
  })

  // 检查是否触发预警
  const quota = await this.prisma.tenantAiQuota.findUnique({
    where: { tenantId_providerId: { tenantId, providerId } },
  })

  if (quota) {
    const remainingPercent =
      (quota.totalQuota - quota.usedQuota) / quota.totalQuota * 100

    if (remainingPercent <= quota.alertThreshold) {
      // 发送预警通知
      await this.sendAlert(tenantId, providerId, remainingPercent)
    }
  }
}

/**
 * 发送预警通知
 */
async function sendAlert(
  tenantId: string,
  providerId: string,
  remainingPercent: number
) {
  // 记录预警日志
  await this.prisma.aiAlert.create({
    data: {
      tenantId,
      providerId,
      remainingPercent,
      status: 'pending',
    },
  })

  // TODO: 发送邮件/短信通知机构管理员

}
```
