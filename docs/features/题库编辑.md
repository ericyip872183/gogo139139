## 一、功能概述

题库编辑功能支持教师和管理员创建、编辑、管理考试题目。系统采用**富文本编辑器 + 媒体资源管理**的混合架构，满足多样化题目内容展示需求。

### 核心特性

| 特性 | 说明 |
|------|------|
| 富文本编辑 | 基于 TinyMCE，支持格式化文本、图片、表格、公式等 |
| 媒体资源管理 | 支持图片/视频/音频上传，租户隔离存储 |
| 多题型支持 | 单选题、多选题、判断题、填空题 |
| 分类管理 | 树形分类结构，支持多级嵌套 |
| 批量操作 | 批量导入/导出/删除 |

## 二、技术架构

### 2.1 数据模型

```prisma
// 题目表
model Question {
  id           String            @id @default(cuid())
  tenantId     String
  categoryId   String?
  type         QuestionType      // SINGLE | MULTIPLE | JUDGE | FILL
  difficulty   Difficulty        // EASY | MEDIUM | HARD
  content      String            // 富文本 HTML 内容
  explanation  String?           // 题目解析
  score        Float             // 分值
  isActive     Boolean
  options      QuestionOption[]
  mediaItems   QuestionMedia[]   // 媒体资源关联
  createdAt    DateTime
  updatedAt    DateTime
}

// 题目选项表
model QuestionOption {
  id         String   @id @default(cuid())
  questionId String
  label      String   // A/B/C/D 或填空序号
  content    String
  isCorrect  Boolean
  sortOrder  Int
}

// 题目媒体资源表
model QuestionMedia {
  id         String   @id @default(cuid())
  questionId String
  type       String   // image | video | audio | file
  url        String   // 文件访问路径
  caption    String?  // 说明文字
  sortOrder  Int
  fileSize   Int?     // 文件大小（字节）
  duration   Int?     // 音视频时长（秒）
  createdAt  DateTime
}
```

### 2.2 文件存储结构

```
uploads/
└── questions/
    └── {tenantId}/
        ├── {uuid}.{ext}    # 图片/视频/音频文件
        └── ...
```

### 2.3 技术选型

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端编辑器 | TinyMCE v6 | Vue 3 组件集成 |
| 后端框架 | NestJS | 文件上传 + 静态服务 |
| 文件处理 | Multer | multipart/form-data 解析 |
| 文件命名 | uuid v4 | 全局唯一文件名 |

## 三、API 接口

### 3.1 题目管理

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/questions` | 题目列表（分页 + 搜索） |
| GET | `/api/questions/:id` | 题目详情 |
| POST | `/api/questions` | 新增题目 |
| PATCH | `/api/questions/:id` | 编辑题目 |
| DELETE | `/api/questions/:id` | 删除题目 |
| DELETE | `/api/questions/batch` | 批量删除 |

### 3.2 媒体资源

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/api/questions/:id/media` | 上传媒体文件 |
| GET | `/api/questions/media/:id` | 获取媒体详情 |
| DELETE | `/api/questions/media/:id` | 删除媒体资源 |

### 4.3 媒体上传逻辑

```typescript
async function handleMediaUpload(file: any) {
  const formData = new FormData()
  formData.append('file', file.file)

  if (!editId.value) {
    ElMessage.warning('请先保存题目后再上传媒体文件')
    return
  }

  const result = await questionsApi.uploadMedia(editId.value, formData)
  mediaItems.value.push(result)
}
```

## 六、常见问题

### Q1：图片上传后无法显示？
- 检查 `uploads` 目录是否有写入权限
- 验证 `main.ts` 中静态资源配置正确
- 检查文件路径是否包含正确的 tenantId

### Q2：富文本编辑器不显示？
- 确认 TinyMCE 相关 CSS/JS 资源加载成功
- 检查 `language: 'zh_CN'` 语言包是否存在
- 尝试移除 `:disabled="!!editId"` 属性排查

### Q3：媒体文件删除后数据库还有记录？
- 删除操作包含数据库记录和物理文件
- 检查 `removeMedia` 方法中文件删除逻辑
- 确认 tenantId 匹配，路径计算正确
