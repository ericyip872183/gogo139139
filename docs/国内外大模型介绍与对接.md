## 二、火山引擎（豆包大模型）

### 2.1 平台介绍

火山引擎是字节跳动旗下的企业级云服务平台，其「豆包大模型」系列提供语言模型、多模态模型、OCR 识别等多种能力。

| 项目 | 说明 |
|------|------|
| 官网 | https://www.volcengine.com/ |
| 大模型文档 | https://www.volcengine.com/docs/ark |
| 控制台 | https://console.volcengine.com/ark |
| API 端点 | `https://ark.cn-beijing.volces.com/api/v3` |

### 2.2 服务特点

- 高性能：依托字节跳动基础设施，响应速度快
- 低成本：相比其他主流大模型，价格更具竞争力
- 中文优化：针对中文场景深度优化
- 多模型：提供多种规格模型

### 2.3 模型列表与价格

#### 语言模型

| 模型名称 | 模型 ID | 上下文 | 输入价格 | 输出价格 | 适用场景 |
|----------|---------|--------|----------|----------|----------|
| Doubao Lite | `doubao-lite-4k` | 4K | ¥0.0003/千 tokens | ¥0.0006/千 tokens | 简单对话 |
| Doubao Pro | `doubao-pro-4k` | 4K | ¥0.0008/千 tokens | ¥0.0020/千 tokens | 复杂推理 |
| Doubao Pro | `doubao-pro-32k` | 32K | ¥0.0008/千 tokens | ¥0.0020/千 tokens | 长文档 |
| Doubao Pro | `doubao-pro-128k` | 128K | ¥0.0050/千 tokens | ¥0.0130/千 tokens | 超长文本 |

#### OCR 服务

| 服务类型 | 价格 | 说明 |
|----------|------|------|
| 通用 OCR | ¥0.00007/张 | 文字识别 |
| 表格 OCR | ¥0.00014/张 | 表格结构识别 |
| 手写 OCR | ¥0.0001/张 | 手写文字识别 |

### 2.4 技术对接

#### API Key 获取
1. 登录火山引擎控制台
2. 进入「方舟·大模型」服务
3. 创建应用，获取 API Key

#### 请求示例

```bash
curl -X POST https://ark.cn-beijing.volces.com/api/v3/chat/completions \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "doubao-pro-4k",
    "messages": [
      {"role": "system", "content": "你是一个有用的助手。"},
      {"role": "user", "content": "你好"}
    ]
  }'
```

#### TypeScript 调用

```typescript
async function callDoubaoAI(apiKey: string, messages: any[], model: string = 'doubao-pro-4k') {
  const response = await fetch('https://ark.cn-beijing.volces.com/api/v3/chat/completions', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'Authorization': `Bearer ${apiKey}`,
    },
    body: JSON.stringify({ model, messages }),
  })
  return response.json()
}
```

```typescript
async function callTencentAI(secretId: string, secretKey: string, messages: any[], model: string = 'hunyuan-standard') {

  const tencentcloud = require('tencentcloud-sdk-nodejs')
  const HunyuanClient = tencentcloud.hunyuan.v20230901.Client

  const client = new HunyuanClient({ credential: { secretId, secretKey } })
  const response = await client.ChatCompletions({ Model: model, Messages: messages })
  return response
}
```

### B. 统一调用接口设计

```typescript
// 统一的 AI 服务接口
interface AIProvider {
  name: string
  chat(messages: any[], model?: string): Promise<any>
  getModels(): Promise<any[]>
}

// 统一调用
async function unifiedChat(provider: AIProvider, message: string, systemPrompt?: string) {
  const messages: any[] = []
  if (systemPrompt) messages.push({ role: 'system', content: systemPrompt })
  messages.push({ role: 'user', content: message })

  const response = await provider.chat(messages)
  return response.choices?.[0]?.message?.content || response.response || ''
}
```
