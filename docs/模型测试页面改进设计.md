## 一、现状分析

### 1.1 当前问题
- 对话框样式简陋，不像现代聊天界面
- 单条消息展示，无法查看历史对话
- 不支持图片生成功能
- 缺少流式输出体验

### 1.2 改进目标
- 仿 ChatGPT/微信聊天界面，提升用户体验
- 支持多轮对话历史
- 新增图片生成功能
- 支持 Markdown 渲染、代码高亮

  id                    String   @id @default(uuid())

### 4.4 后端服务方法

```typescript
// ai-admin.service.ts

/**
 * 测试图片生成
 */
async generateImage(
  providerId: string,
  modelId: string,
  prompt: string,
  options: {
    size?: string,
    quality?: string,
    style?: string,
    n?: number
  }
) {
  const provider = await this.prisma.aiProvider.findUnique({
    where: { id: providerId },
  })

  if (!provider) {
    throw new Error('服务商不存在')
  }

  if (!provider.supportImageGeneration) {
    throw new Error('该服务商不支持图片生成')
  }

  const startTime = Date.now()
  const endpoint = provider.imageEndpoint || provider.baseUrl + '/images/generations'

  try {
    const response = await fetch(endpoint, {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
        'Authorization': `Bearer ${provider.apiKey}`,
      },
      body: JSON.stringify({
        model: modelId,
        prompt,
        size: options.size || '1024x1024',
        quality: options.quality || 'standard',
        style: options.style || 'natural',
        n: options.n || 1,
      }),
    })

    const duration = Date.now() - startTime

    if (!response.ok) {
      const errorData = await response.text()
      return {
        success: false,
        error: `API 错误：${response.status} - ${errorData}`,
        duration,
      }
    }

    const data = await response.json()
    const images = data.data?.map((img: any) => ({
      url: img.url || img.image?.url,
      revisedPrompt: img.revised_prompt,
    })) || []

    // 计算成本（示例：按张数计算）
    const cost = images.length * 0.044 // 示例价格

    return {
      success: true,
      images,
      duration,
      cost,
    }
  } catch (error: any) {
    return {
      success: false,
      error: error.message,
      duration: Date.now() - startTime,
    }
  }
}
```

## 九、API 调用流程

### 9.1 文本对话流程

```
用户输入 → ChatInput 组件
    ↓
发送请求 POST /ai/admin/models/:id/test
    ↓
等待响应（显示加载动画）
    ↓
接收响应 → 添加到对话历史
    ↓
ChatMessage 组件渲染（Markdown + 高亮）
```

### 9.2 图片生成流程

```
用户输入提示词 → ImageGenerator 组件
    ↓
配置参数（尺寸/质量/风格/数量）
    ↓
发送请求 POST /ai/admin/models/:id/generate-image
    ↓
等待响应（显示进度条）
    ↓
接收响应 → 图片列表渲染
    ↓
用户下载/重新生成
```

## 十、注意事项

### 10.1 安全考虑

- API Key 仅存储在后端，不暴露给前端
- 图片下载需验证登录态
- 提示词过滤敏感内容

### 10.2 性能优化

- 大图片懒加载
- 对话历史分页加载（最多保留 50 条）
- 流式输出减少首字等待时间

### 10.3 兼容性

- Chrome 89+ / Edge 89+
- 移动端响应式适配（可选）
