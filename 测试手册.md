## 五、题库管理测试

> 菜单：**题库管理**（用教师账号或超管账号）

### 5.1 新增分类

1. 左侧分类树 → 点「+」→ 名称填 `中医基础理论`，保存
2. 选中该分类 → 再点「+」→ 名称填 `八纲辨证`（子分类），保存
3. **预期**：树显示两级分类

### 5.2 新增题目（单选）

1. 点「新增题目」
2. 填写：
   - 题型：单选题
   - 难度：中等
   - 分值：2
   - 分类：八纲辨证
   - 题目内容：`以下属于八纲之一的是`
   - 选项 A：`阴阳` ✅（选为正确答案）
   - 选项 B：`气血`
   - 选项 C：`脏腑`
   - 选项 D：`经络`
3. 保存
4. **预期**：右侧题目列表出现该题

### 5.3 新增判断题

1. 新增题目 → 题型选「判断题」
2. 内容：`舌红苔黄为热证表现`
3. 选项自动变为「正确/错误」，选「正确」为答案
4. 保存

### 5.4 搜索筛选

1. 在搜索框输入 `八纲` → 点搜索
2. **预期**：只显示包含「八纲」的题目
3. 题型下拉选「判断题」→ **预期**：只显示判断题

## 常见问题

| 现象 | 原因 | 处理 |
|------|------|------|
| 登录提示「机构编码不能为空」 | 没填机构码字段 | 确认填了 `panlei` |
| 点菜单提示 401 | Token 过期 | 重新登录 |
| 考试状态一直是「已发布」没变进行中 | 开始时间未到 | 把开始时间改为当前时间之前 |
| 学生看不到考试 | 没有加入该组织或未指定 | 检查考试的「指定组织」配置 |
